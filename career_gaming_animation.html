    Where creativity meets technology to bring interactive experiences to life. This page will guide you through various career paths, day-to-day responsibilities, software, skills, and projects in the fields of game development and animation.

    Job Types

    Game Designer

    Salary Range: $60,000 - $110,000 per year

    A game designer is responsible for conceptualizing the gameplay, mechanics, and overall structure of a game. They create rules, define objectives, and design levels to ensure an engaging experience for players.

    Animator

    Salary Range: $55,000 - $100,000 per year

    An animator brings characters, environments, and objects to life by creating sequences of motion. They work in 2D or 3D to produce animation for games, films, or digital media, ensuring smooth transitions and realistic movements.

    Game Programmer

    Salary Range: $70,000 - $120,000 per year

    A game programmer writes the code that makes a game functional. They implement gameplay mechanics, physics, and artificial intelligence, ensuring everything runs smoothly and efficiently across various platforms.

    Game Artist

    Salary Range: $50,000 - $90,000 per year

    A game artist designs the visual elements of a game, such as characters, environments, textures, and objects. They work closely with game designers to create the look and feel that aligns with the game’s theme and style.

    Activities/Tasks

    Prototyping Ideas

    Developing early-stage versions of gameplay ideas and mechanics to test player experience and gameplay balance. This includes building small, functional parts of the game to evaluate their viability.

    Animating Character Movement

    Designing character rigs and animating movements like running, jumping, or facial expressions to ensure smooth interactions within the game world or film scenes.

    Implementing Gameplay Features

    Turning game design specifications into functional gameplay elements, including coding systems for player controls, AI behaviors, or multiplayer functionality.

    Creating 3D Models

    Using tools like Blender or Maya to create characters, objects, and environments that align with the game’s artistic direction, ensuring they are optimized for performance and aesthetics.

    Tools + Skills Used

    Unity & Unreal Engine

    Game engines used for building and prototyping game concepts, testing gameplay mechanics, and designing levels. These platforms allow game designers to quickly iterate on ideas.

    Maya/Blender

    Industry-standard tools for 3D animation and modeling. Animators use these to create rigs, animate characters, and produce realistic movements for games or films.

    C# / C++

    Programming languages commonly used in game development. Game programmers utilize these to code core gameplay features, optimize performance, and manage AI systems.

    Photoshop & Substance Painter

    Essential tools for creating and texturing 2D and 3D assets. Game artists use these programs to design textures, environments, and detailed visual elements for characters and objects.

    Getting-Started Project Ideas

    2D Platformer Game

    Create a simple 2D platformer using a game engine like Unity. Focus on designing basic mechanics like jumping, collecting items, and avoiding obstacles. This project will introduce you to core game design principles and prototyping.

    Character Animation Cycle

    Develop a basic animation cycle for a character, including walking, running, and jumping animations using Maya or Blender. This project helps animators practice rigging, keyframing, and smoothing character movements.

    Basic AI for Enemy Characters

    Program simple enemy AI for a game where enemies can follow, attack, and retreat from players. Use languages like C# or C++ in Unity or Unreal Engine to implement basic decision-making and pathfinding.

    3D Environment Design

    Design a small 3D game environment using Blender or Unreal Engine. Focus on creating textures, models, and lighting to create a visually compelling space. This project helps game artists practice level design and optimization for game engines.
